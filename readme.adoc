= S3 For Kotlin - Minio

image:https://img.shields.io/badge/jvm-1.8-blue[title="Compatible with JVM 1.8"]

Minio implementation of the https://github.com/VEuPathDB/lib-s34k[S34k] API.

.*Add to project*:
[source, kotlin]
----

----
